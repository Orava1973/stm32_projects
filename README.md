\# STM32 Projects



This repository contains STM32 projects based on STM32CubeIDE.



\## Base Template



`stm32\_template\_tim\_irq` is a base template project:

\- STM32 Nucleo-F103RB

\- TIM2 used as system tick (2 ms)

\- Event-driven architecture

\- Minimal logic inside interrupts



\## Workflow



\- `main` branch is protected

\- Do NOT modify template directly

\- Create a new branch for each project

\- Copy template as a base for new development



\## Tools



\- STM32CubeIDE

\- Git / GitHub
